# Iris Species Classification

Machine Learning project developed with Streamlit.

## Description

This project classifies Iris flower species using Machine Learning.

The application allows users to:

- Visualize the dataset.
- Analyze graphs and metrics.
- Predict flower species.
- Display a 3D visualization.

## Technologies

- Python
- Streamlit
- Scikit-learn
- Plotly
- Pandas
